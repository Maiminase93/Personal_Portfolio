Sobre mí

Domino HTML, CSS, JavaScript, Git / GitHub, Bootstrap, y próximamente ampliaré mis conocimientos con PHP y WordPress. He trabajado en proyectos colaborativos educativos, como una web para una editorial ficticia y otra en homenaje a las mujeres que han contribuido a la informática.

Lo que más me apasiona del desarrollo web es conectar elementos a través de JavaScript y solucionar problemas de código, ya sea en lógica o en estructura de CSS.

Valoro el trabajo en equipo, la comunicación y la paciencia, y aunque puedo construir webs con facilidad, siempre aprecio las opiniones de los demás, especialmente en el diseño web.

Estoy listo para dar el siguiente paso en mi carrera como desarrollador web.

Experiencia

Desarrollo de una web informativa sobre mujeres en la informática (Proyecto académico grupal)

Colaboré en el diseño e implementación de una página web dedicada a visibilizar a mujeres destacadas en el ámbito tecnológico. Utilicé HTML, CSS y JavaScript para crear una experiencia interactiva, incorporando animaciones dinámicas mediante event listeners y manipulación del DOM. Aprendí a combinar conocimientos adquiridos en clase con investigación autónoma para resolver retos técnicos.

Creación de un sitio web para una editorial ficticia (Proyecto académico en pareja)

Junto a una compañera, desarrollé un sitio web funcional para una editorial ficticia. Me enfoqué en la maquetación responsiva con HTML y CSS y la integración de elementos visuales atractivos. El proyecto fomentó el trabajo en equipo y la gestión colaborativa del código.
